# BBS
JavaとMySQLを用いた電子掲示板です。

## Description
投稿、投稿者による投稿内容の編集と投稿者による投稿の削除を行うことができます。

## Requirement
* Java11
* JavaEE6
* Tomcat9
* MySQL8
